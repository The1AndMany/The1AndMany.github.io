NOTE TO INTERNET EXPLORER USERS:
IE processes most monospace fonts in such a way that the formatting of this website will be displayed incorrectly. I've set a font that doesn't do this to be used by default, but if that font can't be displayed this site will not look its best.
You can check whether the formatting is correct by hitting Ctrl+A and checking whether every line (except possibly the last one) is the same width. If the lines are of different widths the formatting is incorrect; please consider using an alternative browser such as Firefox or Google Chrome.
